🐍 Python Debugger Setup
⚙ VS Code — launch.json Configuration ▶
1

Install Python Extension

Open VS Code Extensions (Ctrl+Shift+X) and install ms-python.python by Microsoft.

2

Create .vscode/launch.json

Press F5 or go to Run → Add Configuration → Python. The file is auto-created under .vscode/launch.json

3

Without Arguments (Simple Script)

JSON — launch.json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Python: Current File",
      "type": "python",
      "request": "launch",
      "program": "${file}",
      "console": "integratedTerminal",
      "justMyCode": true
    }
  ]
}
4

With Command-Line Arguments

JSON — launch.json with args
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Python: With Args",
      "type": "python",
      "request": "launch",
      "program": "${workspaceFolder}/app.py",
      "args": ["--host", "localhost", "--port", "8080", "--debug"],
      "console": "integratedTerminal",
      "env": {
        "PYTHONDONTWRITEBYTECODE": "1",
        "APP_ENV": "development"
      },
      "justMyCode": false
    }
  ]
}
5

Attach to Running Process (debugpy)

Note: Add this to your Python code first: import debugpy; debugpy.listen(5678); debugpy.wait_for_client()
JSON — Attach Mode
{
  "name": "Python: Attach 5678",

  "request": "attach",

  "pathMappings": [{ "localRoot": "${workspaceFolder}", "remoteRoot": "." }]
}
➤ Terminal — pdb / debugpy CLI Debugging ▶
Bash
# Method 1: Run with pdb from terminal
python -m pdb app.py

# Method 2: Add breakpoint inside code
# import pdb; pdb.set_trace()   ← old style
breakpoint()   # Python 3.7+

# Then run normally
python app.py

# pdb commands:
# n → next line
# s → step into function
# c → continue
# l → list source
# p var → print variable
# q → quit
Bash — debugpy
# Install debugpy
pip install debugpy

# Run and wait for VS Code to attach
python -m debugpy --listen 5678 --wait-for-client app.py

# Run WITHOUT waiting for client (non-blocking)
python -m debugpy --listen 5678 app.py
Bash — pdb with arguments
# pdb with script arguments
python -m pdb app.py --host localhost --port 8080

# debugpy with arguments
python -m debugpy --listen 5678 --wait-for-client \
  app.py --host localhost --port 8080

# Parse args in your script
import argparse
parser = argparse.ArgumentParser()
parser.add_argument('--port', type=int, default=8080)
args = parser.parse_args()
◆ Inline API / Module Debugging (debugpy code) ▶
Python
import debugpy

# Start debug server on port 5678
debugpy.listen(("0.0.0.0", 5678))

# Pause until VS Code attaches
print("Waiting for debugger attach...")
debugpy.wait_for_client()
print("Debugger attached!")

# Your code here
def main():
    x = 10
    y = 20
    breakpoint()   # breakpoint here
    result = x + y
    print(result)

main()
✔ Tip: Use PYTHONBREAKPOINT=0 env variable in production to disable all breakpoint() calls safely.

  -jar app.jar

# Run WITHOUT suspension (non-blocking)

  -jar app.jar

jdb -attach localhost:5005

mvn spring-boot:run -Dspring-boot.run.jvmArguments="-Xdebug \
  -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=5005"

from flask import Flask

app = Flask(__name__)

@app.route("/")
def index():
    return "Hello from Flask!"

    parser.add_argument("--host",  default="127.0.0.1", help="Host to bind")
    parser.add_argument("--port",  type=int, default=5000, help="Port number")

from flask import request

@app.route("/api/users")
def get_users():
    page   = request.args.get("page",   1,    type=int)
    limit  = request.args.get("limit",  10,   type=int)
    search = request.args.get("search", "")
    return {"page": page, "limit": limit, "search": search}

Python — Flask routes for path vs query params
# Path param — /api/users/42
@app.route("/api/users/<int:user_id>")
def get_user(user_id):
    return jsonify({"id": user_id})

# Query param — /api/users?id=42
@app.route("/api/users")
def search_users():
    uid = request.args.get("id", type=int)
    return jsonify({"id": uid})
◆ Inline Debugger Methods — pdb, breakpoint, Werkzeug ▶
Python — pdb / breakpoint()
from flask import Flask, request, jsonify
import pdb

app = Flask(__name__)

@app.route("/api/debug-test", methods=["POST"])
def debug_test():
    data = request.get_json()

    breakpoint()    # ← execution pauses here in terminal
    # OR: pdb.set_trace()

    result = data["value"] * 2
    return jsonify({"result": result})

# pdb commands at the prompt:
# n  → next line      s  → step into
# c  → continue       q  → quit
# p data              → print variable
# pp data             → pretty-print
# l  → list source    ll → longer list

Python — structured logging for debugging
import logging
from flask import Flask, request

app = Flask(__name__)

# Configure logging
logging.basicConfig(
    level=logging.DEBUG,
    format="%(asctime)s [%(levelname)s] %(name)s: %(message)s"
)
logger = logging.getLogger(__name__)

@app.route("/api/users", methods=["POST"])
def create_user():
    data = request.get_json()
    logger.debug("Received payload: %s", data)

    try:
        result = process(data)
        logger.info("User created: %s", result)
        return result, 201
    except Exception as e:
        logger.error("Error creating user: %s", e, exc_info=True)
        return {"error": str(e)}, 500
